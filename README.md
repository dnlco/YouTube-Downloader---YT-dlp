This project is a Python-based Graphical User Interface (GUI) for the yt-dlp command-line tool. It allows you to download YouTube videos and audio tracks easily.
Features

    Video Download: Best available quality.

    Audio Extraction: Convert to MP3, WAV, or M4A formats.

    Clipboard Support: Paste URLs with a single click.

    Folder Selection: Choose where to save your files.

    Progress Bar: Real-time feedback on download percentage and status.

    Cancel/Stop: Terminate the download process at any time.

Prerequisites

To run this application, you need:

    Python 3.x installed.

    yt-dlp library:
    pip install yt-dlp


FFmpeg (Required for audio conversion):

        Download and add it to your system PATH, or install via package manager (e.g., brew install ffmpeg or sudo apt install ffmpeg).

How to Use

    Run the script:
    python yt_gui-en.py



Copy a YouTube video URL.

    Click the "Paste from Clipboard" button.

    Select your preferred download folder and format (Video or Audio).

    Click "DOWNLOAD".
